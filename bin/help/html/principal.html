Tabla de Peliculas

    BOTONES
    • Buscar : Este boton filtrara los datos de la tabla y solo mostrara las que su titulo comienze por el texto indicado en el cuadro de texto de la izquierda.
    • Ver Datos: Este boton abrira una ventana con los datos de la pelicula seleccionada en la tabla.
    BARRA DE MENU
    • Archivo : Este menu contiene opciones relacionadas con la aplicaicon
      • Reconectar : Esto realizara una reconexion con la Base de Datos indicada en la de la aplicacion
      • Salir : Esto se encargara del cerrado seguro de la aplicacion
    • Configuracion : Este menu contiene las opciones de configuracion de la aplicacion
      • Conexion : Este abrira una ventana de la de la conexion con la base de datos
      • Salir : Esto se encargara del cerrado seguro de la aplicacion
    • Ayuda : Este menu contiene ayuda sobre la aplicacion
      • Ver Ayuda :Abre lo que estas leyendo ahora mismo
      • Idioma :Te permite cambiar el idioma de la aplicacion entre las opciones mostradas
      • BD: "NOMBRE DE LA BD": Esto indica el nombre de la Base de Datos a la que la aplicacion esta conectada
